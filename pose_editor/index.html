OpenPose Skeleton Editor — drag any joint, click "save PNG" to export

Two ways to move a joint:
A. Drag a colored circle directly (hit area ~14px)
B. Click a row below to "arm" it -> next canvas click jumps it there

Note on overlapping joints: if two joints overlap (e.g. l-hip + r-hip), use Method B — pick from list, then click the new spot on canvas.

OpenPose is 2D-only — no front/back depth. Simple poses fine; for crossed limbs (arms folded behind back) use the 3D editor at zhuyu1997.github.io/open-pose-editor/

Saved PNG -> Downloads folder -> copy to pose_skeletons/pose_<action>_<frame>.png
